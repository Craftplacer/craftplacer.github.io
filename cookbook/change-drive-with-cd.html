Automatically change drive with the 'cd' command in Cmder

From: cmd - Command prompt won't change directory to another drive - Stack Overflow

Add cd=cd /d $* to [Cmder]/config/user_aliases.cmd

Best viewed with Firefox 2+ on 1024x768+

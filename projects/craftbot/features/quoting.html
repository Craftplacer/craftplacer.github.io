Quoting

How to enable

To use quoting, first enable it by typing cb!user features enable quoting.

How to use

Message links

To quote a message, include a link of the message you want to quote.

Imageboard style

It is also possible to quote messages by their ID alone, by typing >>{MESSAGE_ID}.

CraftBot will only check the current channel where the message was posted in.

Behavior

CraftBot will only quote the first message you write.

It is also not possible of retrieving messages from servers, that CraftBot is not in.
